私について

石嵜　克行（いしざき　かつゆき）

千葉県出身の46歳です。
大学卒業後は機械設計エンジニアにおいて、半導体成膜装置をはじめとした産業機械など様々な製品の設計を手掛けてきました。
趣味は車でドライブしたり、買い物に行ったり、関東近辺にキャンプや車中泊をしたりすることです。
機械設計の仕事では、精密さと正確さを求められる業務に携わり、論理的な思考力と細部にこだわる姿勢を養ってきました。
これらの経験は、どのweb系職種にも直結する重要なスキルだと考えています。
機械設計の分野で培ったスキルと経験を活かし、IT業界での新たな挑戦に臨みたいと考えています。

私に出来ること

デザイン

課題の本質を様々な視点から徹底的に分析し、その上で筋道を立てたデザインを行うことを心がけています。細部に妥協せず、良いものを作るためにこだわりを持って取り組んでいます。また、情報設計や機能性といった部分にも配慮し、関連する業界や似たテイストのデザインを、常に最新の知識として蓄積しています。

使用ツール

Photoshop / Illustrator / XD / Figma

コーディング

HTMLやCSSを使用し、デザインを正しく再現します。またJavaScriptを使用しサイトに動きを加えることも可能です。アニメーションによって世界観を演出し、よりWEBデザインの魅力を引き上げられるような実装を目指しています。コードには規則性があり、誰が見てもわかりやすいコーディングを心がけています。

使用言語・ツール

HTML(HTML5,Living Standard) / CSS3 / Javascript / Jquery / Github / Visual Studio Code / Dreamweaver / Atom / SublimeText

これまでの経歴

  • 2000.04

    千葉工業大学工学部精密機械工学科を卒業後、
    昔からモノづくりに深い興味を抱いていた為、日立系の半導体製造装置メーカーで
    未経験から半導体成膜装置の搬送機構の機械設計エンジニアとして働き始める。

  • 2004.10

    半導体業界の経営不振による人員削減の為、会社都合で
    日立系の半導体製造装置メーカーを退社した後、産業用駆動機器メーカーに入社し、
    大手光学機器メーカー向けの半導体露光装置用ダイレクトドライブモーターの製品開発案件を経験する。

  • 2009.06

    リーマンショックによる景気悪化の影響で事業縮小に伴い、会社都合で
    産業用駆動機器メーカーを退社した後、選別機械メーカーに入社し、
    JA農協やリサイクル業者向けの選別機械の製品開発案件を経験する。

  • 2017.05

    選別機械の売り上げの低迷や仕入れ先が次々と倒産し赤字経営となり民事再生法適用により人件費削減のため,収入が激減したことにより再喝が苦しくなったため家族と相談の上、自己都合で選別機械メーカーを退社した後、建築金物・生活雑貨メーカーに入社し、主に生活雑貨にて一般消費者向けの通販カタログ商品である室内物干しの製品開発案件を経験し、意匠デザインに興味を持ち始める。

  • 2019.08

    これまでの経歴や退職理由を踏まえ、今後のキャリアを見つめ直した結果、製造業ではなくIT業界でのクリエイティブ職への強い憧れを抱くようになり、Web制作などを本格的に学ぶためにスクールへの入学を検討し、その準備としてProgate・ドットインストール・Udemy・デイトラを活用し、独学での学習を開始する。

  • 2023.08

    材料費や物価の高騰などにより見積提出後のお客様企業からの失注が多く、その結果開発業務が閉鎖されるため退職勧奨を受け建築金物・生活雑貨メーカー退社。
    その後、Web制作の独学での学習は継続しつつ、技術系派遣会社に入社し、大手自動車メーカーからの受託案件に取り組んでいる企業に配属され、国内外の工場にて車両組立工程に携わる生産技術を経験する。

  • 2024.03

    派遣契約が終了となり、次の案件がなかったので退社。その後、Web制作の独学での学習は継続し、 別の技術系派遣会社に入社し、大手電機メーカーの配電盤を製造している工場に配属され、過去に納品した配電盤の改造対応に携わる。

  • 2024.07

    インターネットアカデミー新宿AVENUE校に入学。IoTエンジニア育成コースを受講中。所属していた技術系派遣会社が派遣契約が終了となり、次の案件がなかったので退社。
